控制流 ​

if 表达式 ​

  • 代码中的条件 必须 是 bool 值
  • 使用过多的 else if 表达式会使代码显得杂乱无章 最好重构代码 (match)
rs
let number = 6;

if number % 4 == 0 {
    println!("number is divisible by 4");
} else if number % 3 == 0 {
    println!("number is divisible by 3");
} else if number % 2 == 0 {
    println!("number is divisible by 2");
} else {
    println!("number is not divisible by 4, 3, or 2");
}

// 在 let 语句中使用 if
let condition = true;
let number = if condition { 5 } else { 6 };
// if 分支和 else 分支的结果都是 i32 整型
// 如果它们的类型不匹配，则会出现一个错误
let number = if condition { 5 } else { "six" };
// error[E0308]: `if` and `else` have incompatible types
// 变量必须只有一个类型。Rust 需要在编译时就确切的知道 number 变量的类型

使用循环重复执行 ​

Rust 有三种循环：loop、while 和 for

loop ​

rs
loop {
    println!("again!");
}

// 循环中的 continue 关键字告诉程序跳过这个循环迭代，并转到下一个迭代

// 从循环返回值
// 如果将返回值加入你用来停止循环的 break 表达式，它会被停止的循环返回
fn main() {
    let mut counter = 0;

    let result = loop {
        counter += 1;

        if counter == 10 {
            break counter * 2;
        }
    };

    println!("The result is {result}"); // The result is 20
}


// 循环标签：在多个循环之间消除歧义
// 如果存在嵌套循环，break 和 continue 应用于此时最内层的循环。
// 可以在一个循环上指定一个 循环标签（loop label），然后将标签与 break 或 continue 一起使用，使这些关键字应用于已标记的循环而不是最内层的循环
fn main() {
    let mut count = 0;
    'counting_up: loop {
        println!("count = {count}");
        let mut remaining = 10;

        loop {
            println!("remaining = {remaining}");
            if remaining == 9 {
                break;
            }
            if count == 2 {
                break 'counting_up;
            }
            remaining -= 1;
        }

        count += 1;
    }
    println!("End count = {count}");
}

while 条件循环 ​

rs
fn main() {
    let mut number = 3;
    // 当条件为真时，使用 while 循环运行代码
    while number != 0 {
        println!("{number}!");

        number -= 1;
    }

    println!("LIFTOFF!!!");
}
fn main() {
    let mut number = 3;
    // 当条件为真时，使用 while 循环运行代码
    while number != 0 {
        println!("{number}!");

        number -= 1;
    }

    println!("LIFTOFF!!!");
}

使用 for 遍历集合 ​

rs
// 使用 for 循环遍历集合中的元素
let a = [10, 20, 30, 40, 50];
for element in a {
    println!("the value is: {element}");
}

// Range，它是标准库提供的类型，用来生成从一个数字开始到另一个数字之前结束(不包括结束)的所有数字的序列
// 1..=4 (包括结束4)
// rev，用来反转 range
for number in (1..4).rev() {
    println!("{number}!"); // 3!   2!   1!
}
// 使用 for 循环遍历集合中的元素

match ​

match 表达式由 match 关键字、用于匹配的值和一个或多个分支构成，这些分支包含一个模式和在值匹配分支的模式时运行的表达式：

rs
let x = 42;

match x {
    0 => {
        println!("found zero");
    }
    // 我们可以匹配多个值
    1 | 2 => {
        println!("found 1 or 2!");
    }
    // 我们可以匹配迭代器
    3..=9 => {
        println!("found a number 3 to 9 inclusively");
    }
    // 我们可以将匹配数值绑定到变量
    matched_num @ 10..=100 => {
        println!("found {} number between 10 to 100!", matched_num);
    }
    // 这是默认匹配，如果没有处理所有情况，则必须存在该匹配
    _ => {
        println!("found something else!");
    }
}
